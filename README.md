
# Edge detection web app

## Edge detection of a given image via OpenCV


### Types of Edge detection implemented :

    1. Sobel Edge detection
    2. Canny Edge detection


### Libraries used : 

    1. OpenCV
    2. Numpy
    3. Streamlit
    4. Pillow (PIL Python Imaging Library)
